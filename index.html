Sobre meu RA

O que é?

  RA é a sigla utlizada para o termo Registro Acadêmico. Número de identificação de cada aluno matriculado. Cada aluno possui o seu próprio, ou seja, não existem RA's idênticos em uma única instituição.

Para que serve?

  Como credencial, o RA serve para que se possa associar no sistema todas os eventos referentes àquele aluno e principalmente para que se consiga acessar portais acadêmicos.

O que fazer se não encontrar meu RA?

  Contate a secretaria para verificar se o seu CPF está corretamente cadastrado!
